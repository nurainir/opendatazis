Cara Berkontribusi

Dalam pengumpulan data tahap pertama, data yang dikumpulkan berupa data utama yaitu alamat panti asuhan yatim piatu, jompo, masjid dan lembaga ZIS. Waktu pengumpulan data adalah tanggal 20 September - 20 Oktober 2011
  1. kirim alamat panti asuhan yatim piatu, jompo, lembaga ZIS ke form berikut ini
    Form Panti Asuhan dan Werda link
    Form Masjid link
    Form Lembaga Amil link
  2. atau jika banyak data yang anda dapatkan, anda bisa kirim email opendatazis@gmail.com dengan attachment bentuk CSV(format file yang biasanya berisi data dimana data tersebut dipasahkan dengan koma atau titik koma. File CSV bisa dibuka lewat editor biasa maupun spreadsheet seperti Excel, Open Office Calc)

    format Panti Asuhan/Werda :
    Nama,Tahun Berdiri,Alamat,Telpon,Kota,Pa/Pi/Camp,Jumlah L,Jumlah P
    Contoh :
    Nama,Tahun Berdiri,Alamat,Telpon,Kota,Pa/Pi/Camp,Jumlah L,Jumlah P Panti Anak Ceria,2000, Jl. Mawar 1, 0213456789,Depok, Camp, 20, 30 Panti Yatim Putri, Jl Melati 2, 0311234567, Surabaya, Pi, 0,40
    download contoh template file disini

    format Masjid
    Nama,Tahun Berdiri,Alamat,Telpon,Kota
    Contoh :
    Nama,Tahun Berdiri,Alamat,Telpon,Kota
    Baiturrahman, 1990, Jl Bantaran II, 034134455, Malang
    download contoh template file disini

    format Lembaga Amil
    Nama,Tahun Berdiri,Alamat,Telpon,Kota,Keg1,Keg2,Keg3 Contoh :
    Nama,Tahun Berdiri,Alamat,Telpon,Kota,Keg1,Keg2,Keg3
    Dompet yatim, 2005, Jl. kanan kiri, 022345678, Bandung, beasiswa, santunan modal kerja, pembangunan sekolah
    download contoh template file disini
    Keterangan : jika lebih dari 3 tambahkan keg4, keg5 dst
  3. Data diatas dapat anda peroleh dari internet misal data panti asuhan dari pantiasuhan.info, forum zakat dan lain-lain.

Pengumpulan data tahap kedua akan dilakukan tahun depan setelah seluruh data dari tahap pertama dipublikasikan. Adapun data yang dikumpulkan adalah data Muzakki per kecamatan seluruh Indonesia. Untuk itu perlu adanya kerjasama dengan Baznas.
Selain pengumpulan data, anda dapat membantu dengan memakai data tersebut dalam berbagai keperluan seperti meneliti hubungan tingkat kesejahteraan dengan jumlah amil dan jumlah penduduk, melakukan visualilasi dengan google map tingkat persebaran panti seluruh Indonesia dan lainnya.
